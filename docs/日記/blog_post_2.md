該說些什麼呢？從好幾年前就想考微軟的認證，不過，那時除了專業技術的缺乏，也包括英文能力的不足，更因為自已的毅力不夠，無法自我學習，所以一直都裹足不前，沒有實際行動。  直到過去二年，由華經離職後，我總算用了二年半的時間在圖書館，默默的讀了以下幾門科目，完成了考試。  

- 70-536 Microsoft .Net Framework Application Fundation 2.0
- 70-515 Web Applications Development with Microsoft .Net Framework 4
- 70-461 Querying Microsoft SQL Server 2012
- 70-462 Administering Microsoft SQL Server 2012 Databases
- 70-463 Implementing a Data Warehouse with Microsoft SQL Server 2012

通過這些科目，得到三個 LOGO，哈！還真貴。  覺得自已做事太沒有效率了，也看出來自已自退伍後是在過什麼樣的生活，雖然很累很辛苦，但是卻因為沒有目標，沒有計畫，日子就這樣一天天的過去，  到頭來，因為底子不夠，準備這些考試才會用這麼久的時間。  

![](http://127.0.0.1:8080/_images/private/LOGO_MCP_70536.png)  ![](http://127.0.0.1:8080/_images/private/LOGO_MCTS_70515.png)  ![](http://127.0.0.1:8080/_images/private/LOGO_MCSA_SQL2012.png)  

這段日子的心得是什麼，我真的覺得，我很沒用，做事很沒毅力，日子過的太安逸了。  我就這樣子過了大半個人生，我什麼都沒有，真的是一事無成，沒朋友，沒家庭。  

我不想多說什麼，期待自已，不要再抱怨了，幾點原則給自已：  

- 每年年初要列出自己的年度計畫，年中要檢視進度，年終要檢討進度。
- 做事情要計畫，計畫立了，就要堅定意志去完成。
- 不要再去想過去不好的人事物，那些都已經無法改變。
