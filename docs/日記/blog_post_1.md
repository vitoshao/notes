前幾天看到保哥說他為了參加IT鐵人大賽，在一個月時間寫完了３０篇文章，而這些文章的字數統計下來居然超過一百萬字。  今天好奇我的這個BLOG倒底有多少字，所以統計了一下 BlogContent 這個欄位的總長度，結果是 3,348,637 個字。  這其中當然包含 Html Tag ，所以，唉，要加油了。  

列個表，以後隨時就來統計一下吧。

#### 執行語法
```sql
Use [Vito2012]
select 
'日期' = convert(char(10), getdate(), 111)
,'篇數' = count(1)
,'字數' = replace(convert(varchar(100),cast(sum(LEN(blogcontent))  as money), 1),'.00','') 
from tbldocuments
```
