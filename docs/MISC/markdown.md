## 文字與段落

這是一段文字，可以使用 `**` 或 `__` 來加粗，例如：**這是加粗的文字**。__這也是是加粗的文字__	

這是另一段文字，段落之間可以使用 `空行` 來分隔。

也可以使用 `*` 或 `_` 來斜體，例如：*這是斜體的文字*。_這也是是斜體的文字_

刪除線可以使用 `~~` 來標記，例如：~~這是刪除線的文字~~
